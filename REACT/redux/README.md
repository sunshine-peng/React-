# 总结 redux的设计使用原则
1. store 是唯一的
2. 只有store能改变自己的内容
3. reducer是一个纯函数

# redux的核心api
1. createStore 创建store
2. store.dispatch 派发action
3. store.subscribe 监听store数据的改变
4. store.getState 获取store里面的所有数据
